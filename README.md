# HTML Tools

Hi, I'm Gold, and I am a fledging, self-taught software engineer with a love for JavaScript and automation. I've created a collection of web-based tools that could be found anywhere, but I either wanted them to function a certain way and/or just wanted to sharpen my skills and post some proofs-of-concept. :thumbsup: <br><br>

### My tools:  

#### :heavy_check_mark: Checklist Maker -
A super simple checklist maker with 2 functions. Create a printable blank(save for title) checklist of (x) rows with (y) checkboxes per row, or type out each item and add/remove checkboxes individually for more granular control.  

#### :page_with_curl: Simple List Maker -
Essencially a copypasta of Checklist Maker; sans checkboxes.
